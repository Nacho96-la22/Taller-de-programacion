# _Taller de programación_ 📚
## 📁 Estructura del Repositorio
Este repositorio contiene la practica completa de taller de programación organizados por años. Incluye soluciones de prácticas, parciales y trabajos prácticos de las materias de programación.

## ☝🏻🤓 Detalles importantes 
-  **📆 2023:**
    
    **Modulo objeto:** el archivo _jre-8u381-windows-x64.exe_ es el instalador de Java y el archivo _PaqueteLectura.jar_ es importación de modo lectura.

    **Modulo concurrente:** el archivo _r-Info3.1.jar_ es el programa de r-info (por ahora tengo esa versión y quizá que ellos tienen la ultima versión actualizada) que se ejecuta directamente sin instalación.
